# Code of Conduct

## Our standard

Participation in this project should be harassment-free for everyone. Contributors are expected to be respectful, direct, and constructive.

Examples of expected behavior:

- giving actionable technical feedback
- assuming good faith while reviewing code and ideas
- respecting different levels of experience
- focusing criticism on the work, not the person

Examples of unacceptable behavior:

- harassment, intimidation, or personal attacks
- discriminatory language or conduct
- deliberate disruption of discussions or reviews
- publishing someone else's private information without permission

## Scope

This Code of Conduct applies in project spaces, issue threads, pull requests, discussions, and related community communication.

## Enforcement

Project maintainers may remove comments, reject contributions, or block participation for behavior that violates this Code of Conduct.

## Reporting

Report conduct issues privately to the project maintainer instead of escalating in public issue threads.
